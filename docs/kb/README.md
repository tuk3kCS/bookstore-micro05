# Knowledge Base (KB)

This folder contains the knowledge base documents used by the advisor RAG service.

## Authoring format
- Use Markdown files (`.md`)
- Put a short YAML frontmatter header at the top:

```yaml
---
title: Shipping policy
tags: [shipping, policy]
updated_at: 2026-04-06
---
```
